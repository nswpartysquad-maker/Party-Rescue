# Party Rescue SEO Website

This is a proper Next.js multi-page website with separate URLs:
- /
- /packages
- /gallery
- /contact

## Run locally
npm install
npm run dev

## Deploy on Vercel
1. Import this folder into Vercel.
2. Add your domains:
   - partyrescue.com.au
   - www.partyrescue.com.au
3. Set DNS:
   - A record for @ -> 76.76.21.21
   - CNAME for www -> cname.vercel-dns.com

## Notes
- Pricing is intentionally removed.
- Gallery uses Instagram reel embeds.
- Meta titles, descriptions, and canonicals are added for SEO.
